# RoadSense - Smart Road Monitoring

This project detects road damage using OpenCV and visualizes it on Google Maps.

## Features
- Real-time camera detection
- Damage reporting via JSON
- Maps visualization (if API key provided)

## Instructions
1. Compile the C++ code using OpenCV
2. Run `road_sense.exe` to detect damage
3. Open `map.html` to visualize locations (API key optional)
